สร้าง เว็บตัวช่วยสร้างระบบ Merge ให้แล้วครับ — เป็นไฟล์ HTML ไฟล์เดียว ทำงานฝั่งเบราว์เซอร์ล้วน (ไฟล์ของผู้ใช้ไม่ถูกส่งขึ้นเซิร์ฟเวอร์ ปลอดภัย) เปิดในเบราว์เซอร์ได้เลย หรือจะเอาไปโฮสต์ก็ได้
ผู้ใช้ทำ 3 ขั้น

อัปโหลด Excel (.xlsx) + Word (.docx) ที่วางฟิลด์ {{ชื่อฟิลด์}}/MERGEFIELD ไว้ + เลือกโหมด (ใช้ในชีต / Web App)
กดตรวจไฟล์ → ระบบอ่านฟิลด์จาก Word (รวมหัว/ท้ายกระดาษ) เทียบกับหัวคอลัมน์ Excel แล้วรายงาน:

✅ ตรง — ฟิลด์กับคอลัมน์ตรงเป๊ะ
🔵 แก้ให้ — ต่างแค่ช่องว่าง/สะกดใกล้เคียง → ใส่ "นามแฝงฟิลด์" ให้อัตโนมัติ
🔴 เพิ่มคอลัมน์ — ไม่มีคอลัมน์รองรับ → เพิ่มคอลัมน์ให้ในไฟล์ Excel
ℹ️ ไม่ได้ใช้ — คอลัมน์ที่ไม่มีฟิลด์ในเอกสาร


กดสร้างระบบ → ดาวน์โหลด .zip ที่มี: Code.gs, Index.html, MergeDialog.html, ไฟล์ Excel ที่จัดระเบียบแล้ว (เพิ่มชีต แม่แบบ/ตั้งค่า/นามแฝงฟิลด์ + คอลัมน์แม่แบบ + นามแฝงอัตโนมัติ), ไฟล์ Word ที่ปรับ run ให้ {{ }} ต่อกัน (รูปแบบเดิม), และ README พร้อมขั้นตอนติดตั้ง+ใช้งาน (ปรับตามโหมดที่เลือก) — หน้าเว็บก็สรุปวิธีติดตั้งให้ด้วย

สองแบบตามที่ขอ — เลือกโหมดได้ (แบบใช้ในชีตผ่านเมนู/แผงปุ่ม กับแบบ Web App ที่ผู้ใช้ได้ URL ไปใช้) โดย .zip ใส่ไฟล์ครบทั้งสองแบบ ต่างกันแค่คำแนะนำการติดตั้ง สลับภายหลังได้
ทดสอบจริงครบวงจรแล้ว: ป้อน Excel (มีคอลัมน์ต่างจากฟิลด์ Word แบบมีช่องว่าง) + Word ตัวอย่าง → ระบบสร้างนามแฝงให้ → เมื่อนำผลลัพธ์ไป merge จริง {{ชื่อผู้ลงนาม}} แทนค่าถูกต้องผ่านนามแฝง ทุกฟิลด์เต็ม ไม่มีตกค้าง และได้ Word แท้
